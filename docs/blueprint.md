## Core Features:

- Structure de base du layout: Mettre en place une structure de mise en page fondamentale en utilisant Tailwind CSS pour obtenir un design responsive.
- Configuration Vite: Configurer Vite afin de bénéficier d’un bundling de modules rapide et efficace, ainsi qu’un environnement de développement performant.

## Style Guidelines:

- Couleur principale : Violet (#4a48aa) pour une base simple.
- Couleur d’arrière-plan : Gris clair (#F0F0F0), presque blanc, pour mettre en valeur les autres couleurs.
- Couleur d’accent : Vert doux (#A3E4D7) pour signaler la réussite ou un retour positif.
- Police  des titres : *Lato*, sans-serif, pour une apparence moderne, technique, objective et neutre.
- Police du corps : *Lexend*.
- Mise en page simple et épurée, avec un minimum de distractions.
- Aucune animation ou transition au départ.
